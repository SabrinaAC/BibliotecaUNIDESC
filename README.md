# 📚 Biblioteca UNIDESC

Sistema de gerenciamento de biblioteca desenvolvido como projeto acadêmico utilizando **ASP.NET Core MVC**, **C#**, **Entity Framework Core** e **SQL Server**.

## 📖 Sobre o Projeto

O objetivo deste sistema é facilitar o gerenciamento de uma biblioteca, permitindo o cadastro, edição, consulta e exclusão de livros, além de oferecer uma interface administrativa moderna e intuitiva.

O projeto está sendo desenvolvido seguindo o padrão **MVC (Model-View-Controller)** e utilizando boas práticas de desenvolvimento e versionamento com Git e GitHub.
